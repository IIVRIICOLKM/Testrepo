## 사용설명서

### 요구 사향
- python 3 and Interpreter
  
### 패키지 설치
```
pip install -r requirements.txt
```
### 실행
```
git clone https://github.com/IIVRIICOLKM/NFA_DFA_Transfer/
python nfatranser.py > outfile.txt
```
